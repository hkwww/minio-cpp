
# minio-cpp

该仓库针对[minio-cpp SDK](https://github.com/minio/minio-cpp)，并不依赖vcpkg进行编译。

## 使用方法

**安装curlpp**

```bash
  git clone https://github.com/jpbarrette/curlpp.git
  cd curlpp
  cmake ..
  make -j
  sudo make install
```

**安装minio-cpp**

```bash
  cd minio-cpp
  cmake ..
  cmake ..
  sudo make install
```
